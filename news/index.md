# Changelog

## qesR 0.4.4

- Added
  [`get_qes_master()`](https://thomasgareau.github.io/qesR/reference/get_qes_master.md)
  for harmonized merged datasets across QES studies.
- Added merge quality controls: within-year respondent de-duplication
  and all-empty-row removal.
- Added broader harmonization for age, age group, education, turnout,
  vote choice, and weights.
- Improved codebook workflows with compact/wide/long layouts and
  value-label extraction helpers.
- Added researcher website content with merged-dataset documentation,
  study citations, and analysis examples.

## qesR 0.4.3

- Added `get_question(..., full = TRUE)` support for richer
  question-text recovery.
- Improved codebook metadata parsing and handling of support files.

## qesR 0.4.2

- Expanded QES catalog entries and Dataverse file selection behavior.

## qesR 0.4.1

- Added
  [`get_decon()`](https://thomasgareau.github.io/qesR/reference/get_decon.md)
  and export updates.

## qesR 0.4.0

- Added codebook retrieval and formatting helpers.
